# **App Name**: DosisPaws

## Core Features:

- Physiological Constants: Display physiological constants such as temperature, heart rate and respiratory rate for different animal species, fetched from an internal database.
- Fluid Therapy Calculator: Calculate fluid replacement for veterinary patients, and maintenance dosages based on dehydration, maintenance, and ongoing losses, taking the animal's weight into account.
- Drug Dosage Calculator: Calculate drug dosages for different animal species based on weight and selected medication, using a fuzzy search tool over the internal drug database. Generates and shows the correct dose and route of administration. Incorporates and accounts for varying drug concentrations, including tablets and liquids.
- Nutritional Calculator: Calculates Resting Energy Requirement (RER) and daily caloric needs for animals based on weight and physiological state.
- Transfusion Calculator: Calculates the correct volume of blood to transfuse based on patient and donor hematocrit levels, along with patient weight.
- Gestation Calculator: Calculates gestation timelines for different animal species based on the breeding date.
- Vademecum AI Summary: Using the medication information database, and the user's queries and requests as context, generate tailored clinical information summaries and client instructions for each medication using a tool. Provides a better and easier method of searching medications based on patient data.

## Style Guidelines:

- Primary color: Soft lavender (#D0B4FF) to convey trust and veterinary care.
- Background color: Dark gray (#212121) to provide contrast and readability.
- Accent color: Teal (#4DB6AC) for highlights and action items to create a clear visual hierarchy.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent and clear icons from Lucide-React for different functions and categories.
- Use a clean and intuitive layout with clear sections for each calculation and data display.
